Getting Started

1. Download or clone the repository

>> git clone git@github.com:raulsanchez1024/generator-MERN.git

2. Pull templates folder out of mernstarter

3. Git Initi

>> git init

3. Install all dependancies

>> npm i
>> cd frontend
>> npm i

3. Replace MongoURI & SecretKey

module.exports = {
  mongoURI:"",
  secretOrKey: ""
};
E.G.,
module.exports = {
  mongoURI: "mongodb://raul:123456S@ds12fd234234892.mlab.com:25892/mernstarter",
  secretOrKey: "wussgooodddd"
};

4. Start the application from templates folder

>> npm run dev

Heroku Deployment

1. Install Heroku CLI

2. Create new Heroku app

>> heroku create

4. Add to Heroku

>> heroku git:remote -a HEROKU-APP-NAME
>> git add .
>> git commit -am "make it better"
>> git push heroku master
>> heroku open

React Custom Components

<PrivateRoute />

This component will be used in the App.js file when using routes. Simply use this route with any route you want to be protected. Pass the component name as props.

<Switch>
 <PrivateRoute path="/dashboard" exact component={Dashboard} />
</Switch>
