Getting Started

1. Download or clone the repository

>> git clone git@github.com:raulsanchez1024/generator-MERN.git

2. Install all dependancies

>> cd mernstarter/app/templates/
>> npm i
>> cd mernstarter/app/templates/client
>> npm i

3. Replace MongoURI & SecretKey

module.exports = {
  mongoURI:"",
  secretOrKey: ""
};
E.G.,
module.exports = {
  mongoURI: "mongodb://raul:123456S@ds12fd234234892.mlab.com:25892/mernstarter",
  secretOrKey: "wussgooodddd"
};

4. Start the application

>> cd mernstarter/app/templates/ >> npm run dev

Heroku Deployment

1. Install Heroku CLI

2. Create new Heroku app

>> heroku create

3. Push to git

4. Add to Heroku

>> heroku git:remote -a HEROKU-PROJECT-NAME
>> git push heroku master
>> heroku open
